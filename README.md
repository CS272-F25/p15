# Car Dealership Website

A small, simple fake car dealership website. We allow you to search for cars, compare them to each other, find dealership locations, and schedule a test drive

## Overview
- **Purpose:** Help users browse inventory, compare vehicles, find locations, and schedule a test drive, trying to emulate a real car dealership website
- **Data:** Data fetched from the carqueryapi according to user options

## Pages
- `index.html` — Home page with some featured vehicles and site navigation.
- `inventory.html` — The main page to search for cars.
- `compare-cars.html` — Compare cars you found on the inventory page.
- `our-locations.html` — Addresses and information about dealership locations.
- `schedule-test-drive.html` — Form to schedule a test drive.
